# CRUD NOTES APP

This is a demo app for writting notes and checking the progress of what you wrote.

As a Demo we developed just to complete all CRUD flow.

Endpoints will be added along the project is developed.

Tecnology stack:

- Python
- Fastapi
- SQLAlchemy
- Pydantic -> For data validation

How to run the proyect?

```bash
cd backend
uvicorn main:app --reload --port 8080
```
